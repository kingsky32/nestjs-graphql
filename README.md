## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## .env Example

```text
# JWT
JWT_ACCESS_TOKEN_SECRET=secretKey
JWT_ACCESS_TOKEN_EXPIRATION_TIME=1800000
JWT_REFRESH_TOKEN_SECRET=secretKey
JWT_REFRESH_TOKEN_EXPIRATION_TIME=17280000000

# APP
APP_ENV=development
APP_NAME=App
APP_URL=http://localhost:4000
APP_PORT=4000

# GRAPHQL
GRAPHQL_DEBUG=true
GRAPHQL_PLAYGROUND=false

# THROTTLER
THROTTLER_TTL=60
THROTTLER_LIMIT=10

# DATABASE_POSTGRES
DATABASE_POSTGRES_HOST=POSTGRES_HOST
DATABASE_POSTGRES_PORT=POSTGRES_PORT
DATABASE_POSTGRES_USERNAME=POSTGRES_USERNAME
DATABASE_POSTGRES_PASSWORD=POSTGRES_PASSWORD
DATABASE_POSTGRES_DATABASE=POSTGRES_DATABASE

# AWS
AWS_ACCESS_KEY_ID=AWS_ACCESS_KEY_ID
AWS_ACCESS_KEY_SECRET=AWS_ACCESS_KEY_SECRET
AWS_S3_REGION=AWS_S3_REGION
AWS_S3_BUCKET_NAME=AWS_S3_BUCKET_NAME

# MAIL
MAIL_SERVICE=MAIL_SERVICE
MAIL_HOST=MAIL_HOST
MAIL_AUTH_USER=MAIL_AUTH_USER
MAIL_AUTH_PASS=MAIL_AUTH_PASS
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
